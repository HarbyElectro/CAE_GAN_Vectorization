# Documentation

This folder contains research documentation for the `CAE_GAN_Vectorization` repository.

The goal of this folder is to make the project easier to understand, reproduce, extend, and cite.

## Files

- `datasets.md` — dataset descriptions, expected folder structures, and usage notes.
- `experiments.md` — recommended experiment design and training workflows.
- `model_architectures.md` — overview of the CAE, GAN-AE, and Video Autoencoder architectures.
- `latent_vector_format.md` — explanation of latent-vector formats, storage types, and usage.
- `metrics.md` — explanation of reconstruction, similarity, and GAN discriminator metrics.
- `reproducibility.md` — guidelines for reproducible research experiments.
